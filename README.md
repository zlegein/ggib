## GQL SERVER

#### Install node 16.x or greater

#### Install yarn

```bash
yarn
```

#### Start the App 
```bash
  yarn dev
  ```

GQL server runs on http://localhost:4000/graphql


## UI Application

#### Install node 16.x or greater

#### Install yarn

```bash
  yarn
  ```

#### Start the App 
```bash

  ```
